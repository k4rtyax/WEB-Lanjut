- Konfigurasi Template NiceAdmin
    1. Sebelumnya, sudah berhasil membuat view baru untuk menu home, produk, dan keranjang. Namun masih berupa halaman  yang hanya berisi teks dan link sederhana. Tampilan pada Codeigniter yang terletak pada folder Views pada dasarnya adalah berisi code HTML, sehingga bisa dipercantik menggunakan CSS yang dibuat sendiri, ataupun menerapkan template web yang sudah jadi. 
    Pada project kali ini, akan menerapkan template jadi yang siap pakai bernama NiceAdmin. Template ini sudah cukup lengkap dan bagus, terdapat tampilan responsive untuk layar desktop, tablet, dan mobile. Live demonya bisa dicoba di https://bootstrapmade.com/demo/NiceAdmin/

Kemudian extract dan letakkan folder **NiceAdmin** di dalam folder **public** yang ada pada project.

- Template Slicing untuk membuat Layout
    1. Langkah selanjutnya adalah, code html pada file app\Views\layout.php akan dipisah-pisah (slicing) menjadi component header, sidebar, footer, dan content terpisah.
    
    Tujuan melakukan slicing ini, supaya mempermudah jika akan melakukan perubahan tampilan. Misalnya jika akan mengubah daftar menu yang ada pada sidebar, cukup mengubahnya pada file app\Views\components\sidebar.php, maka semua file view yang menggunakan layout yang sama (layout.php) akan langsung bisa menampilkan menu sidebar yang terbaru.
    
    Umumnya, pada file template, terdapat penanda pada setiap section tampilannya. Misalnya pada header, sebagai penanda pada awal bagian terdapat tulisan 
    **<!-- ======= Header ======= -->**
        
        ![image.png](attachment:d1e4f871-be55-4759-bdf8-c50666ee8eef:image.png)
        
        Dan sebagai penanda akhir bagian terdapat tulisan
        
        **<!-- End Header -->**
        
        ![image.png](attachment:25f5f295-7ff7-409d-89a1-aa2462f3e2c1:image.png)
        
        Penanda pada setiap bagian-bagian seperti ini sangat membantu developer saat melakukan slicing template. 
        
        Selanjutnya yang harus dilakukan kemudian adalah pada file app\Views\layout.php, copy baris 43-270 dan paste di file app\Views\components\header.php

- Judul yang dinamis menggunakan url helper
    1. Kemudian selanjutnya, judul pada setiap halaman akan dibuat dinamis sesuai dengan URL menu yang sedang dibuka. Judul terletak pada file layout.php bagian card-title. Jadi, misal sedang buka halaman Produk (http://localhost:8080/produk) maka judul yang tampil juga tertulis “Produk”.
        
        
        Hal ini bisa dilakukan dengan menggunakan salah satu helper yang disediakan oleh CI yaitu URL helper. Caranya, aktifkan dulu helper url dengan cara buka file app/Config/Autoload.php dan tambahkan pada variable **$helper**. https://codeigniter.com/user_guide/helpers/index.html 
        
        ![image.png](attachment:0c0d27b8-0193-4b83-9239-be60b4803a4c:image.png)
        
        Autoload adalah fitur yang memungkinkan untuk memuat library, helper, dan model secara otomatis setiap kali sistem berjalan, sehingga tidak perlu memuatnya secara manual di setiap controller atau file.
        
        ![Untitled](https://prod-files-secure.s3.us-west-2.amazonaws.com/332d3749-1e40-47dd-9041-59670fa067ee/3b78642c-005d-4dde-abc1-da929bdcc881/Untitled.png)
        
        Helper url diaktifkan karena akan menggunakan fungsi **uri_string()**, fungsi ini akan mencetak url setelah base_url. Base_url adalah url untuk root project (utama), konfigurasinya terdapat pada file app\Config\App.php
        
        ![image.png](attachment:70d41e1b-2b64-45a9-a683-d9b4ed38e048:image.png)
        
        Contoh, misalnya ketika user mengakses url http://localhost:8080/produk diharapkan pada bagian judul content akan tercetak juga kata **produk.**
        
        Gunakan fungsi **uri_string()** pada file layout.php, jika fungsi **uri_string()** ada isinya maka akan tampil sesuai urlnya, tapi jika tidak ada isinya maka akan tampil tulisan home.
        
    2. Terapkan pada file layout, buka file app/Views/layout.php dan tambahkan code berikut pada baris paling atas
        
        ```php
        <?php
        $hlm = "Home";
        if(uri_string()!=""){
          $hlm = ucwords(uri_string());
        }
        ?>
        ```
        
        Ini untuk mengambil path dari url untuk menentukan halaman apa yang sedang diakses oleh user. Disimpan dalam variable $hlm.
        
        ![Untitled](https://prod-files-secure.s3.us-west-2.amazonaws.com/332d3749-1e40-47dd-9041-59670fa067ee/d0145c9a-fb81-4629-8604-e0db47b88a8c/Untitled.png)
        
        Kemudian gunakan nilai variable $hlm untuk title halaman, ini akan tampil pada judul tab di browser.
        
        ```html
        <title>- Toko - <?php echo $hlm ?></title>
        ```
        
        ![image.png](attachment:611530d6-d6d1-4424-b795-f2f078de6ea1:image.png)
        
        ![image.png](attachment:1199a62f-0195-4955-b086-50fd2bf02613:3ffd42e7-9122-4eda-bb24-5d9581485b3a.png)
